🌍 Roamly – MERN Airbnb Clone

Roamly is a full-stack MERN (MongoDB, Express, React, Node.js) web application inspired by Airbnb. It allows users to browse, create, update, and manage property listings with a modern UI and RESTful backend.

🚀 Features

-🏠 Browse all property listings
-🔍 View detailed listing pages
-➕ Add new listings
-✏️ Update listings
-❌ Delete listings
-⚡ RESTful API integration
-🎨 Responsive UI (React-based)
-🔄 Full CRUD operations
